<h1 align="left">MindTrack – Smart Task & Productivity Tracker</h1>

###

<p align="left">MindTrack is a smart productivity web application designed to help users stay focused, organized, and stress-free. It enables users to manage tasks, track goals, and monitor progress in one unified platform. MindTrack emphasizes clarity, simplicity, and efficiency, making it an ideal companion for students and professionals aiming to improve daily productivity and focus.<br><br>Features:<br>– Task management system to add, update, and track daily tasks<br>– Goal tracking to monitor progress and maintain consistency<br>– Clean and distraction-free user interface for better focus<br>– Responsive design for smooth usage across desktop, tablet, and mobile devices<br><br>Tech Stack: <br>Frontend: HTML, CSS, Bootstrap, JavaScript<br>Backend: Postgre, Django</p>
